virtual void KCloud::Resource::prepareForSend ( )
virtual

Inizializza i dati per poter avviare la trasmissione.

Dopo aver chiamato clear(), setta il numero dei pacchetti m_packets facendo la divisione tra la dimensione totale e quella di un pacchetto (si vedano getNetworkSize() e getBytesPerPacket()), il numero di bytes rimanenti m_spareBytes con il resto della divisione tra i due precedenti, ed infine, inizializza anche il contatore dei bytes m_bytesCounter con il valore oppportuno in base al numero dei pacchetti.
