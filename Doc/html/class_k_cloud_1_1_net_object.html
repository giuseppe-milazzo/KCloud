virtual void KCloud::NetObject::behaviorOnSend ( const qint64  dim)
protectedpure virtualslot

Stabilisce come devono essere gestiti i segnali della socket in scrittura.

Parametri
dimnumero di bytes scritti sulla socket.

Tramite questo slot è possibile tenere il conto di quanti bytes sono stati realmente scritti sulla socket. QTcpSocket offre il segnale bytesWritten(qint64) per cui tramite questo slot è possibile decrementare il numero dei bytes da scrivere (vedi m_bytesCounter) di modo che, quando tutto il pacchetto è stato inviato, si può inviare un nuovo pacchetto tramite changeBlock().

virtual void KCloud::NetObject::prepareForSend ( )
virtual

Inizializza i dati per poter avviare la trasmissione.

Dopo aver chiamato clear(), setta il numero dei pacchetti m_packets facendo la divisione tra la dimensione totale e quella di un pacchetto (si vedano getNetworkSize() e getBytesPerPacket()), il numero di bytes rimanenti m_spareBytes con il resto della divisione tra i due precedenti, ed infine, inizializza anche il contatore dei bytes m_bytesCounter con il valore oppportuno in base al numero dei pacchetti.
